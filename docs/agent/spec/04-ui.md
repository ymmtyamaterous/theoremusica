## 1. デザインシステム

### 1.1 カラーパレット

| 変数名 | カラーコード | 用途 |
|--------|------------|------|
| `--ink` | `#0e0d0c` | 背景色（最も暗い） |
| `--paper` | `#f5f0e8` | テキスト・前景色 |
| `--cream` | `#ede7d8` | サブ背景 |
| `--gold` | `#c9a84c` | メインアクセント |
| `--gold-light` | `#e8cc87` | ホバー時アクセント |
| `--rust` | `#8b3a2a` | 上級レベルバッジ |
| `--sage` | `#4a6741` | 入門レベルバッジ |
| `--muted` | `#6b6356` | 補助テキスト |
| `--staff` | `#2a2520` | 五線譜の線 |

| 種別 | フォント | ウェイト | 用途 |
|------|---------|---------|------|

| 本文 | DM Sans | 300 / 400 / 500 | ナビ, 本文, メタ情報 |

| コンポーネント | 概要 |
|--------------|------|
| `Button` | primary / ghost / outline バリアント |

| `Badge` | レベルバッジ (入門 / 中級 / 上級) |

## 2. レイアウト共通要素

### 2.1 ナビゲーション (`Header`)

```
┌────────────────────────────────────────────────────┐
│  𝄞 Teoria   コース  理論  練習  コミュニティ  [無料で始める] │
└────────────────────────────────────────────────────┘
```

- sticky ヘッダー（スクロール追従）
- 背景: `rgba(14,13,12,0.92)` + `backdrop-filter: blur(12px)`

- ロゴ: Playfair Display イタリック, ゴールド, 音部記号 𝄞
- ナビリンク: 大文字, letter-spacing 0.08em, 未認証時「無料で始める」CTA ボタン, 認証済み時はユーザーメニュー表示

### 2.2 フッター (`Footer`)

```
┌──────────────────────────────────────────────────────┐
│  𝄞 Teoria          © 2026 Teoria     プライバシー · 利用規約 │
└──────────────────────────────────────────────────────┘
```

## 3. 画面別設計

---

### 3.1 トップページ (`/`)

**構成セクション:**

```
[Nav]
[Hero]
[Staff Divider]
[Courses Section]
[Interactive Theory Section]
[Stats Section]
[Footer]
```

#### 3.1.1 Hero セクション

2カラムグリッド:

```
┌───────────────────────┬──────────────────────┐
│                       │                      │
│  [タグ] 音楽の言語を学ぶ │    ┌─五線譜─────┐   │
│                       │    │  ピアノ鍵盤  │   │
│  音楽を               │    └────────────┘   │
│  理解する             │   ♩  ♪  ♫  ♬        │
│  ための場所           │                      │
│                       │                      │
│  [レッスンを始める]   │                      │
│  [デモを見る →]      │                      │
└───────────────────────┴──────────────────────┘
```

**ピアノコンポーネント仕様:**
- 白鍵 7 個（C D E F G A B）, 黒鍵 5 個（C# D# F# G# A#）
- クリックで active 状態（ゴールドに変色）、0.6 秒後に解除
- 浮遊する音符アニメーション（♩ ♪ ♫ ♬、フェードイン/アウト、3秒ループ）

#### 3.1.2 コースセクション

- セクションタグ「カリキュラム」
- タイトル「コースを選ぶ」+「すべて見る →」リンク
- 3カラムグリッドでコースカードを表示

**コースカード仕様:**
```
┌────────────────────────┐
│  [icon]                │
│  [レベルバッジ]         │
│  コースタイトル         │
│  説明文（2行）          │
│  [X レッスン] [約X時間] │
│                      → │
└────────────────────────┘
```
- ホバーで背景色変化 + 矢印表示
- レベルバッジ: 入門=sage, 中級=gold, 上級=rust

#### 3.1.3 インタラクティブ理論セクション

2カラム:
- 左: テキストコンテンツ（五度圏の説明・特徴リスト）
- 右: 五度圏インタラクティブ SVG

**五度圏 SVG 仕様:**
- 外周: 長調キー（C G D A E B F# Db Ab Eb Bb F）
- 内周: 短調キー（Am Em Bm F#m C#m G#m D#m Bbm Fm Cm Gm Dm）
- セグメントホバーでハイライト
- 中心ラベル「五度圏 / Circle of Fifths」

#### 3.1.4 統計セクション

4カラムグリッドのカード:
- `12K+` 学習者が登録
- `98` 個のレッスン
- `4.9` 平均評価（5点満点）
- `∞` 音楽の可能性

### 3.3 コース詳細ページ (`/courses/:courseId`)

```
[Nav]
┌──────────────────────────────────────────────────┐
│  ← コース一覧                                     │
│                                                  │
│  [icon]  コースタイトル          [レベルバッジ]    │
│          説明文                  [受講登録ボタン]  │
│          [X レッスン] [約X時間]                   │
│                                                  │
│  ─────────────────────────────────────────────── │
│  レッスン一覧                                     │
│  ┌───────────────────────────────────────────┐   │
│  │ #1  レッスンタイトル         [完了済み ✓]  │   │
│  │ #2  レッスンタイトル         [15分]        │   │
│  │ #3  レッスンタイトル         [20分]        │   │
│  └───────────────────────────────────────────┘   │
└──────────────────────────────────────────────────┘
[Footer]
```

- 受講登録ボタン: 未登録→「受講を始める」、登録済み→「受講中」（グレー）
- レッスン一覧: 完了済みはチェックマーク表示
- 未認証ユーザーが「受講を始める」クリック → ログインページへリダイレクト

### 3.4 レッスンページ (`/courses/:courseId/lessons/:lessonId`)

> 要認証

```
[Nav]
┌──────────────────────────────────────────────────┐
│  ← コース名                                       │
│  ┌────────────────────────┬────────────────────┐ │
│  │                        │  コース進捗         │ │
│  │  レッスンタイトル       │  ████░░░░  3/12    │ │
│  │                        │                    │ │
│  │  [Markdown コンテンツ] │  レッスン一覧       │ │
│  │                        │  ├ #1 ✓           │ │
│  │                        │  ├ #2 ✓           │ │
│  │                        │  ├ #3 ← 現在      │ │
│  │                        │  ├ #4             │ │
│  │                        │  └ ...            │ │
│  │                        │                    │ │
│  │  [← 前のレッスン]      │                    │ │
│  │           [このレッスンを完了 →]             │ │
│  └────────────────────────┴────────────────────┘ │
└──────────────────────────────────────────────────┘
```

- 左メイン: Markdown レンダリングされたレッスン本文
- 右サイドバー: コース内進捗バー + レッスン一覧（現在のレッスンをハイライト）
- 下部ナビ: 前後レッスンへのリンク + 完了ボタン
- 完了ボタン押下 → `lessons.complete` API 呼び出し → 次のレッスンへ自動遷移（あれば）

## 4. レスポンシブ対応方針

| ブレークポイント | 対応内容 |
|--------------|---------|
| `< 768px` (モバイル) | 1カラム, ハンバーガーメニュー, Hero は縦積み |
| `768px ~ 1024px` (タブレット) | コースグリッド 2カラム |
| `> 1024px` (デスクトップ) | コースグリッド 3カラム, フルレイアウト |
